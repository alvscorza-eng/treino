# Meu Treino no iPhone (instalar como app)

## 1) Publicar rápido com GitHub Pages
1. Crie um repositório no GitHub (por exemplo: `meu-treino`).
2. Envie os arquivos da pasta `Treino` para a raiz do repositório.
3. No GitHub: **Settings > Pages**.
4. Em **Source**, selecione **Deploy from a branch**.
5. Escolha a branch `main` e pasta `/ (root)`.
6. Salve e aguarde o link ficar disponível.
7. Abra no iPhone o link final (sem sufixo de arquivo).

Exemplo:
`https://SEU-USUARIO.github.io/meu-treino/`

## 2) Instalar na Tela de Início (iPhone)
1. Abra o link no **Safari**.
2. Toque no botão **Compartilhar**.
3. Toque em **Adicionar à Tela de Início**.
4. Confirme em **Adicionar**.

Pronto: vai abrir como app, em tela cheia.

## 3) Observações importantes
- Seus dados ficam salvos no navegador do iPhone (armazenamento local).
- Se você apagar dados do Safari, os treinos podem ser removidos.
- Para não perder histórico, posso adicionar exportar/importar backup (JSON) no próximo passo.

## 4) Quando publicar atualização
1. Atualize os arquivos no GitHub Pages e aguarde o deploy.
2. Feche o app da Tela de Início e abra novamente com internet.
3. Se ainda aparecer versão antiga, no Safari abra o link web uma vez e recarregue.
4. Em último caso: iPhone > Ajustes > Safari > Avançado > Dados dos Sites > remova o site e instale de novo na Tela de Início.
